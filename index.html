Evaluation Results

A deliberate trade-off, measured honestly.

OLD pipeline → NEW pipeline · 5 users · gpt-4o

MetricOLDNEW
Entity F131%35%
ObjectsNot supported34% F1
Widget Coverage72%44%
Grounded EvidencePartial100%

What the numbers say

  • Both pipelines are LLM-based — measured on the same GT, same 5 synthetic users.
  • NEW adds objects, context windows, and evidence grounding OLD cannot produce.
  • With gpt-4o, NEW wins composite GT score (+4pp); Gemini 3.5 Flash leads NEW widget coverage (82%).
  • Aggregate across all 6 models: OLD still leads entity F1 (39% vs 35%) and widget coverage (74% vs 50%).

Honest limitations

Small, diverse dataset LLM still wins on raw extraction Relationships + long-term memory = future work
